엔트리로 배우는 AI&DATA

소프트웨어 교육 플랫폼 ‘엔트리’의 인공지능 및 데이터 분석 기능을 활용하여, 다양한 문제를 해결하는 소프트웨어 프로그램을 만들 수 있습니다.

  • 인공지능 활용

    네이버의 인공지능 기술을 통해 인공지능 기술의 다양한 활용 사례를 체험할 수 있는 기능을 제공합니다.

  • 모듈 학습

    머신러닝의 학습 방법 중 하나인 지도학습을 직접 체험하는 기능으로 이미지, 텍스트, 음성 데이터를 분류하는 모델(인공지능)을 학습시키고 결과를 확인할 수 있습니다.

    • 1. 데이터 입력

      모델이 학습할 데이터를 클래스 별로 나누어 입력합니다.

      *클래스: 데이터의 묶음으로 새로운 데이터를 분류하는 기준입니다.
    • 2. 학습

      입력한 데이터로 모델을 학습시킵니다. 모델은 데이터를 동해 특정한 규칙과 패턴 을 찾아가게 됩니다.

    • 3. 확인

      학습을 마친 모델이 새로운 데이터를 올바르게 분류하는지 결과를 확인합니다.

  • 데이터 분석

    데이터를 직접 수집하여 시각화하고 분석해봄으로써 데이터를 기반으로 문제를 해결해보는 경험 을 제공합니다.

    • 데이터 수집

      엔트리에서 제공하는 데이터나 수집한 데이터 셋을 불러오거나, 직접 테이터를 입력하여 테이블로 저장합니다.

      *엔트리에서는 공공 데이터와 네이버 블로그 통계, 코로나19 현 황 등 다양한 기본 데이터셋을 제공합니다.
    • 시각화

      수집한 데이터를 한눈에 파악할 수 있도록 막대, 꺾은선, 원형 등 다양한 그래프로 나타냅니다.

    • 분석

      데이터를 다양한 방법으로 변환해보며 데이터 속에 숨어있는 유용한 정보를 찾고 문제해결 과정에 적용해봅니다.

세상의 다양한 문제를 AI&DATA를 활용하여 해결해보는 경험!

직접 만나보는 네이버의 AI 기술

보고(Image), 듣고(Speech), 읽고(Text), 말하고(Voice), 분석하고 이해하는(NLP) 네이버 클로바의 AI기술을 경험하고, 네이버 랩스의 로봇, 자율주행 등 미래기술을 만나보세요!

    • CLOVA Voice 음성 합성

      자연스러운 고품질의 목소리를 구현하고, 텍스트 정보를 읽어주는 기술입니다.

    • CLOVA Speech 음성인식

      사람의 음성 신호의 파형을 분석하여 사람의 음성을 텍스트로 변환하는 기술입니다.

    • CLOVA AI Call, 고객센터

      음성인식기술과 자연어 처리기술 등을 활용하여 전화를 통한 고객 응대 기능을 지원합니다.

    • CLOVA OCR 문자 판독

      사진 속에서 텍스트 정보를 찾고 의미를 판별하는 기술입니다.

    • CLOVA Vision 이미지 분석

      빅데이터를 기반으로 선생 학습된 모델을 통해 이미지에서 유용한 정보를 추출합니다.

    • CLOVA NLP 자연어처리

      한글 자연어처리 기술을 끊임없이 연구하고 이를 실제 서비스에 적용하며 고도화합니다
